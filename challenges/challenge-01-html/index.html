Relação Client x Server

Em relação ao conceito básico da arquitetura web, quando se acessa algum website, o usuário está fazendo uma requisição (GET) a um servidor, através da internet. Ao receber essa requisição, o servidor como resposta envia arquivos (HTML, CSS, JS, imagens..), necessárias ao site, podendo se comunicar com o banco de dados caso seja necessário.

Estrutura HTML

O HTML possui duas partes principais: cabeçalho (tag head, responsável por carregar arquivos e configurar a página) e corpo (tag body, cujo conteúdo é visível ao usuário). Cada tag possui alguns atributos gerais*, sendo eles:

  1. Atributo id: identificador único de cada elemento, muito utilizado no JavaScript para se comunicar com os conteúdos do HTML.
  2. Atributo class: nome da classe criada no CSS.
  3. Atributo style: recebe como valor o código CSS, responsável pela estilização do conteúdo.

* Atributos não obrigatórios
